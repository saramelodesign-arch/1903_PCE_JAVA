# 1903_PCE_JAVA

Repositório com exercícios de Programação em Java desenvolvidos no âmbito da UFCD de Programação Estruturada (PCE), da formação no CESAE Digital.

Os exercícios estão organizados por fichas práticas e abordam:
- Arrays e matrizes
- Funções e procedimentos
- Leitura e escrita de ficheiros
- Processamento de ficheiros de texto e CSV
- Estruturas de repetição e decisão

## Estrutura do projeto
- `src/` – código Java organizado por fichas práticas  
- `pce/Ficheiros/` – ficheiros de texto e CSV usados nos exercícios  
- `.gitignore` – ficheiros e pastas ignorados pelo Git
